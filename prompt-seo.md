You are an expert SEO engineer, web performance specialist, and frontend developer.

Your task is to analyze and optimize an existing static website (multiple HTML pages) for **maximum SEO performance, discoverability, and page speed**.

---

## 🎯 Goals

- Improve **search engine ranking (Google, Bing)**
- Optimize for **AI search and LLM indexing**
- Enhance **page speed and performance**
- Ensure **clean semantic structure**

---

## 🔍 SEO Optimization Requirements

### 1. Meta Tags (for every page)
- Add unique:
  - <title> (under 60 characters)
  - meta description (under 155 characters)
- Add:
  - meta keywords (optional but clean)
  - author tag
  - robots tag

---

### 2. Open Graph + Social Sharing
- Add OG tags:
  - og:title
  - og:description
  - og:image
  - og:url
- Add Twitter cards

---

### 3. Structured Data (Schema.org)
- Add JSON-LD for:
  - Organization
  - Website
  - Breadcrumb (if applicable)

---

### 4. Semantic HTML
- Ensure proper use of:
  - <header>, <nav>, <main>, <section>, <article>, <footer>
- Only one <h1> per page
- Logical heading hierarchy (H1 → H2 → H3)

---

### 5. Keyword Optimization
- Identify primary keywords for each page:
  - AI Studio
  - Generative AI
  - AI Agents
  - Edge AI
  - LLM Frameworks
- Naturally integrate into:
  - headings
  - paragraphs
  - alt text

---

### 6. Image Optimization
- Add descriptive alt tags
- Use modern formats (WebP if possible)
- Lazy loading for images

---

### 7. Performance Optimization
- Minimize CSS and JS
- Defer non-critical scripts
- Optimize Tailwind usage
- Ensure fast load (<2s ideal)

---

### 8. Internal Linking
- Add links between:
  - Home
  - Solutions
  - About
  - Careers
- Improve crawlability

---

### 9. Sitemap + Robots
- Generate:
  - sitemap.xml
  - robots.txt

---

### 10. URL Structure
- Ensure clean URLs:
  - /about
  - /solutions
  - /careers

---

### 11. Accessibility (SEO boost)
- Add:
  - aria labels where needed
  - proper contrast
  - readable font sizes

---

### 12. AI/LLM Optimization (Modern SEO)
- Add:
  - Clear structured content
  - FAQ section (optional)
  - Concise summaries
- Optimize for:
  - ChatGPT, Claude, Perplexity indexing

---

## 📦 Output Requirements

- Update ALL HTML files:
  - index.html
  - about.html
  - careers.html
  - solutions.html
- Generate:
  - sitemap.xml
  - robots.txt
- Provide optimized code (ready to deploy)

---

## ✨ Tone

- Clean, structured, professional
- SEO-first, performance-first
- Avoid unnecessary fluff

---

Now analyze the website and return optimized versions of all files. Also provide a short SEO audit summary explaining what was improved and why.
